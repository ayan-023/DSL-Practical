Implementing a real-time undo/redo system for a text editing application using a Stack data
structure. The system should support the following operations:
• Make a Change: A new change to the document is made.
• Undo Action: Revert the most recent change and store it for potential redo.
• Redo Action: Reapply the most recently undone action.
• Display Document State: Show the current state of the document after undoing or
redoing an action.

-------------------------------------------------------------------------------------

class TextEditor:
    def __init__(self):
        self.undo_stack=[]
        self.redo_stack=[]
        self.doc=""

    def new_txt(self,txt):
        self.undo_stack.append(self.doc)
        self.doc=txt
        self.redo_stack.clear()

    def undo(self):
        if not self.undo_stack:
            print("Noting to Undo")
            return
        self.redo_stack.append(self.doc)
        self.doc=self.undo_stack.pop()
        print("\nUndone Successfully\n")
        editor.display()
    
    def redo(self):
        if not self.redo_stack:
            print("Noting to redo")
            return
        self.undo_stack.append(self.doc)
        self.doc=self.redo_stack.pop()
        print("\nRedone Successfully\n")
        editor.display()
    
    def display(self):
        print("\nCurrent Document:")
        print(self.doc if self.doc else "[Empty Document]")
